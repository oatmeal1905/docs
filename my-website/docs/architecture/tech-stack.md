# Технологический стек

| Компонент | Технология | Обоснование |
|-----------|-----------|-------------|
| Backend | Python (Django / Flask) | Быстрая разработка, большая экосистема |
| База данных | PostgreSQL | Надёжная реляционная СУБД, поддержка JSONB |
| Frontend | HTML, CSS, JavaScript | Простота, не требует отдельного фреймворка в MVP |
| Внешние API | OpenWeatherMap | Бесплатный тариф, стабильная работа |
| Хостинг | VPS | Полный контроль, низкая стоимость |
| Контроль версий | GitHub | CI/CD, хранение кода и документации |
